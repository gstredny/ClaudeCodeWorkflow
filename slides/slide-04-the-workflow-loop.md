# The 6-Phase Workflow Loop

```
Phase 1  PLAN .............. Claude AI (browser)
   |     Define success criteria, tests, constraints
   v
Phase 2  EXPLORE ........... Claude Code (CLI)
   |     Check tasks, explore codebase, generate plan
   v
Phase 3  REVIEW ............ Claude AI (browser)
   |     Challenge assumptions, simplify, improve
   v
Phase 4  EXECUTE ........... Claude Code (CLI)
   |     Implement. Log every attempt to task file
   v
Phase 5  VERIFY ............ You + Claude Code
   |     Walk Done Criteria one by one
   v
Phase 6  CLOSE-OUT ......... You
         Approve. Retro entry. Move to closed/
```

### Two tools, one loop

- **Claude AI (browser)** = Thinking (plan, review)
- **Claude Code (CLI)** = Doing (explore, execute)

---

**Speaker notes:**
Most tasks follow all 6 phases. Simple bugs might skip Phase 3. Failed verification bounces back to Phase 4. The key discipline: every phase produces an artifact â€” plans are written, attempts are logged, criteria are checked off. Nothing lives only in your head.
